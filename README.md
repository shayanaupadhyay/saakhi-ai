# Saakhi AI

AI-powered multilingual RAG shopping assistant built to improve product discovery on Meesho using Hinglish natural language search.

## Problem

Tier-2 and Tier-3 Indian shoppers often don't know the exact English product keywords, leading to poor search relevance and abandoned sessions.

## Solution

Saakhi AI understands natural Hinglish queries, retrieves semantically relevant products using RAG, and explains why each recommendation matches the user's intent.

## Features

• Multilingual search
• RAG architecture
• Semantic retrieval
• Product recommendations
• Conversational interface
• Explainable AI responses
